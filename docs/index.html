Academic Projects

  • An approach to generate video captions in Bengali with semantic consistency Thesis

    This research focuses on Bengali video captioning. It implements sequence-to-sequence models (LSTM, BiLSTM, GRU) combined with CNN features (VGG-19, Inceptionv3, ResNet50v2) and introduces an attention mechanism for the first time in Bengali video captioning. A novel Bengali dataset is created from the MSVD dataset, and the attention-based model outperforms existing models, setting a new benchmark.

  • Emotion Detection Using EEG for Special Teens

    Developed a system that detects happiness or sadness by analyzing brainwave data collected via an EEG headset. Trained an SVM model on data from 20 special teens, captured through emotion-inducing video sessions.

  • Harry Potter (Blender 3D project)

    This project is a 3D scene created in Blender, depicting the iconic moment from Harry Potter where Harry goes through the wall at Platform 9 ¬æ and sees the Hogwarts Express for the first time.   • Shakrain Festival (An OpenGl 2D project)

    It's a 2D OpenGL simulation of the Shakrain festival, where users can control a kite using the arrow keys (up, down, left, right). Pressing 'n' triggers a day-to-night transition, and during the night, sky lanterns light up the scene, adding to the festive atmosphere. The project provides an interactive and visually engaging experience of the traditional celebration.

  • A smart shopping cart system

    This project features an Arduino-based shopping cart that allows users to scan product barcodes and automatically add items to the cart. It incorporates a weight sensor to verify the product quantity, ensuring accuracy during checkout. Additionally, the system includes an automated billing process, streamlining the checkout experience and reducing wait times in stores.

  • Tic Tac Toe using Arduino

    This project is a Tic Tac Toe game developed using an Arduino microcontroller. It features a TFT display for interactive gameplay, allowing users to play the classic game directly on the screen.
